## Overview
The Farm Management System is a game application with backend storage services that allows users to create, save, and manage farm simulation projects. 

The system consists of:
- Frontend: Phaser-based game (src/scenes/)
- Backend Services: Data import/export, storage via Dexie/IndexedDB
- API Layer: RESTful endpoints for remote access
- CLI Tools: Command-line interface for automation and maintenance

## Getting Started
1. Start the development server: `npm run dev`
2. Access the game at http://localhost:5173
3. Use the CLI tools for data management tasks

For detailed API documentation, see [API Endpoints](./api_endpoints.md)
